# SimpleNotesEditor
音ゲーの譜面を作るシンプルなエディター。mp3形式の音楽ファイルに対応しています。

## Develop Environment
* Visual Studio 2019
* .NET Core 3.1 WPF

## Libraries
* Prism.Unity
* ReactiveProperty
* NAudio
* Newtonsoft.Json
