# SimpleNotesEditor
音ゲーの譜面を作るシンプルなエディター。

## Environment
* Visual Studio 2019
* .NET Core 3.1

## Libraries
* Prism.Unity
* ReactiveProperty
* NAudio
* Newtonsoft.Json
